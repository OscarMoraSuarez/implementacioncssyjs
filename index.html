MANIPULACION DEL DOM

Que es el DOM?

El modelo de objeto de documento (DOM) es una interfaz de programación para los documentos HTML y XML. Facilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido.

En este turorial desarrollare ejercicios sencillos sobre manipulacion del DOM utilizando css y javascript ademas de la implmentacion de fuentes de DAFONT.

Temas de este tutorial

  • cambiar texto de un elemento
  • cambiar estilos y clases de un elemento
  • Crear Nuevos elementos
  • Eliminar Elementos
  • Actualizar listas dinamicamente
